# Issue tracker guide

Use GitHub Issues for bug reports and feature requests.

## Bug report template

```
## Description
[clear description of the bug]

## To reproduce
1. [step 1]
2. [step 2]
3. [step 3]

## Expected behavior
[what should happen]

## Actual behavior
[what actually happens]

## Environment
- OS: [e.g. Linux Mint 21, macOS 14, Windows 11]
- Python version: [e.g. 3.12]
- GhostVPN version: [e.g. v1.2.0]

## Logs
[paste output with --verbose flag]
```

## Feature request template

```
## Problem
[what problem are you trying to solve]

## Proposed solution
[how you want it to work]

## Alternatives considered
[other approaches you've thought about]

## Additional context
[anything else]
```

Open a new issue at https://github.com/20player11/GhostVPN/issues/new.
